## Tech Stack  

- Front-end: **Next JS**, **Tailwind**, **Redux**, **Redux Toolkit Query**, **Material UI DataGrid**, **AWS Amplify**, **AWS API Gateway**
- Back-end: **Node.js**,  **AWS EC2**, 
- Database and storage: **AWS S3**, **Prisma**, **AWS RDS**
